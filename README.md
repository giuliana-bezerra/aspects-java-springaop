## Tecnologias

- [Spring Boot](https://spring.io/projects/spring-boot)

## Como Executar

- Clonar repositório git:
```
git clone https://github.com/giuliana-bezerra/aspects-java-springaop.git
```
- Construir o projeto:
```
./mvnw clean package
```
- Executar:
```
java -jar ./target/aspects-java-springaop-0.0.1-SNAPSHOT.jar
```
